坚持你就已经胜过很多人｜2021年7月的片言只语

2021-08-01

07.01

倾听是非常重要，也需要很多技巧，专注、不轻易打断，与讲者同步，如果在倾听的时候自己云游万里，那不可能是好的倾听者。
不同的话语也只适合不同的人群说出来，因为不同的身份和角色，就会有适合它的语言。我们应该努力让自己受到重视，提高话语权。
有些话能说，有些话不能说。

07.02

在家庭环境中，夫妻间，子女和父母间，自己和子女间，总会有无法达成一致的想法和意见，我觉得大方向就是要求同存异。不要试图让对方认同或者服从自己，这样才能家和万事兴。

07.03

人到周末或假期就容易懈怠，所谓劳逸结合，这样无可厚非。但是那些成长特别快和你感觉牛的人，往往在别人休息娱乐时依然在努力奋斗着。这世界上，大部分的差距轮不到智商来决定，光是勤奋程度已经能很大程度拉大了人与人之间的差距。

07.04

今天和邻居聊天，他目前是一家创业芯片公司驱动组的负责人，大家行业相近，自然会聊到工作事业上的东西。然后年龄相仿，不可避免要说到中年危机。
他应对中年危机的想法是自己创业，但由于行业原因，他们这个行业如果想创业，大部分都需要很重的资本投入和行业资源，对于他这种工薪阶层而言，距离还有点遥远，但是要想做相关的又有前景的，目前他也还在摸索中，表达出了一些焦虑感。他个人其实比较偏向于做行业相关大的产业，说不定能财富自由。
我相信像我们这种过了而立之年的，都会有焦虑感，我自己也不例外。对于他的创业想法，我是提了些想法和建议，对于大部分普通人而言，真的想要创业，还是要抓住自己自身优势和市场需求。尤其是自身优势挖掘有时也是起基础性作用的。
首先自身没有很大的资本，那就尽量找轻资本的事情，财富自由是很多人的梦想，如果觉得某些领域容易财富自由，就往那边走，但是不适合自己也只是一个遥不可及的梦。
现在流行一个词是“精益创业”，先把最简单的商业模式建立起来，跑通闭环，再努力规模化放大，这样的概率更高，挫折感也没那么大。
很多人创业时都会觉得自己会做一份伟大的事业，其实前提先活下去，创造出利润，是普通人创业的必要条件吧。

07.05

今天第一次跑去星巴克办公了一个上午，感觉有点新鲜，总体效率还是不错的。
以前不时会听说一些人带着电脑在星巴克办公，有些还颇有装模作样的感觉。但是这次自己亲历，其实还是比较靠谱的。整个上午我没有消费，早早坐在一个角落里，连接上餐厅的免费wifi，联网体验也不错的。有人可能会觉得嘈杂，但是我戴上耳机，感觉这些杂音更加像白噪音，只要自己专注，其实周围的影响程度很低。
下次有机会还会再去试试，也许还会消费一下，刚好有一张星礼卡代金券可以用，如果你有兴趣去装一下B，可以带着自己的笔记本或者平板电脑去试试。

07.06

自己的电脑背包外层的皮脱皮了，老婆和家人都叫我赶紧换了，我也打算换了，然后他们叫我别省这点钱，赶紧换。
关于消费，我的消费观一直是实用派，几乎不追求品牌或者新鲜华而不实的东西，所以在老婆家人眼里我算是很省的人，他们有时看我有些东西用了很久没换还替我捉急，我自己反倒觉得无所谓，没坏又没什么问题，感觉没太大动力去换。
身边有些朋友同时和我不同，时不时看他们换新手机新电脑，甚至换新车，消费欲望比较大，有时看到别人换了一样东西，才发现自己的好像也用了挺久了，也想起该换一下。这时就会觉得，时不时给自己买点东西，其实也是一种激励，在经济能力允许下，时不时犒劳一下自己，其实让自己更有动力去赚钱，对吧？

07.07

我们人生有四类事情：
一、自己擅长的事
二、自己喜欢的事
三、对别人有价值的事
四、能赚钱的事
人生的终极目标应该是能做一件事，同时具备上面四种属性。

07.08

最近在朋友圈里很多人在做一个社交类的项目，类似的项目自己去年操作过，虽然也赚过点小钱，但是总体而言纯粹是为了拿经验。综合下来看，还是不太适合自己做。要做成这种项目，自身拥有资源的才比较好做，例如你是大 V，有一批粉丝，又或者你手上有流量，可以做冷启动，又或者你没有流量，但是有能力和有流量大人合作，那也成。说到底还是一个流量转化变现的问题。
这个项目除了要有流量，也要用微商那套逻辑来运营和拉新，说实话并不是自己的强项和想要走的路径。虽然说如果能赚钱又何必自命清高，但是我自知不合适，为啥还要投入这个机会成本是吧，有了之前的一些经验，现在自己看项目的感觉和逻辑也发生了变化。

07.09

今天在看一本科普漫画《漫画病菌、人类与历史》，是混子曰团队出品的，觉得挺好，趣味之余也达到了科普的目的。我们如今获取信息的形式有多种，文字、图文、漫画、视频、音频等。纯粹的文字和图文适合比较深入的学习和反复咀嚼的内容，而漫画、视频、音频这些形式的内容更容易接受，如今短视频大行其道，就是因为它更符合大众的需求。
我们做内容时，先想清楚自己的受众和自己的内容，根据内容和受众来选择形式，达到最佳的效果。例如比较深度的知识就适合用图文，简单、轻松面向大众的则适合用漫画、视频、音频等形式。其中音频还有一个重要的特点，就是可以多线程进行，你开车、跑步、做饭都可以听音频，这个特点也是其他形式所不具备的。

07.10

今天和女儿还有她同学一家到外面玩，大家后面一起玩了夹娃娃游戏，我运气爆发，仅夹了三次就夹中了一个，简直喜出望外，印象中这是我这辈子最快夹到的了。另一边厢，她同学爸爸一直不停的夹，估计夹了十多遍才夹到，然后后来继续追加了几遍又夹到一个。
其实我知道夹娃娃游戏本身是一种概率游戏，大部分机器的夹子都是会有一个松动的开关，这个开关可以通过概率控制松紧，大部分时间夹子都会比较松，即便你夹得非常准，升起的一瞬间它会是会掉下来。
商家就是利用这个概率来保证了自己的利润期望是正的。可以说夹娃娃机是充分利用了人性，带有一定的赌博成分，就算有时你明知会夹不起来，还是控制不住想继续夹，因为它总会有一两次能让你夹紧，给你带来一点成功喜悦感。

07.11

虽然说搜索引擎存在已经很多年了，但是你会发现其实大部分人其实并不会搜索，甚至不知道有搜索这种东西。所以你常常会看到有人在各种场合问一些很简单无脑的问题，这些问题只需搜索一下就会出来答案。有些人是懒，有些人是没意识，无论是哪样，他们都错失了获取信息的一个重要手段。
即便有人会自己主动搜索，但也仅仅会一个简单的百度输入关键词，不会筛选结果，不会用合适的关键字，更不会用正确的搜索渠道，所以，学会搜索，你已经能超越大部分人。

07.12

情绪不佳的时候，就自己一个人冷静一下，在外面走走，换个心情。

07.13

如果你想了解心理咨询实际上是怎样的，建议可以读一下《蛤蟆先生看心理医生》，这边书比较有意思，用拟人的手法给读者展现了心理咨询时患者与医生分别是怎样的，作为患者，首先要坦诚和信任，同时还不能把这个想成“药到病除”，咨询的过程是一个开解和辅导的过程，医生会尽最大的努力让患者从心理疾病中摆脱出来，而患者自身也要积极配合和信任，通过自己的努力走出困境。

07.14

《被讨厌的勇气》这本书可以说给我带来了一些震撼和新的认知。里面核心的「目的论」让人有拍案叫绝的惊喜。我们很多人感觉不幸福，觉得自己没法改变了，现实复杂残酷，自己充满了不幸。这些底层是因果论在作祟。认定过去决定了现在和未来，而过去已经无法改变，所以现在和未来也无法改变了。但事实上，现在和未来怎么样，取决于我们怎么看待它和用什么态度去面对。

07.15

根据阿德勒的「目的论」，自卑这种性格情绪不是与生俱来或者外部因素导致的，一个人自卑，其实是自己内心选择的结果，因为自卑，他可以逃避人际关系，这样就可以避开人际关系中很多让自己难受的问题，人就是通过自卑这个借口，让自己有理由可以不面对很多事物人物。

07.16

说来可能见笑了，今天第一次去星巴克自己堂食，用的生财有术中奖的星礼卡，为了用满30减5的券，点了个32的卡布奇诺和打包了一份小咖啡蛋卷。说实话咖啡并没有什么吸引我的，香味不是那么吸引，喝到后面几乎有点喝不下，有点苦涩。感觉星巴克就是典型的品牌增值案例，饮品本身并不是非常特别，但是品牌做得好，所以可以卖得比较贵。

07.18

自认性格属于内向的，不太擅长社交，也发现其实大部分人都是内向的，真正外向的人其实属于少数。就如杨涛涛哥这种给人感觉情商极高的人，也自认是内向。虽然如此，但是他说即便内向，但是外在的表现却是可以通过后天努力改变的，他自身就是这样的例子。
还有一点就是，随着年龄增大，会发现身边的知心朋友越来越少，除去精力、时间的原因，还有一个是方式技巧的原因，他认为最有效的一个办法就是多请客吃饭，真心的那种，对于想要维系的关系，多多请客吃饭，这是最好的方式，请客吃饭吃不穷人，但是带来的收益却很大。

07.19

我觉得人们都会说书中自有黄金屋，事实的确如此，但是我觉得光是把书放在那里人已经有一种莫名的安全感，如果你感到焦虑，多读点书，如果实在读不了，买来放在那里也会感到充实，难怪会有汗牛充栋的故事。

07.20

自己用印象笔记也挺久了，以前没有太深度使用，大部分时间拿来存一些资料。今天发现了一个课程，教如何利用印象笔记为自己的外挂大脑。这里面的一个核心点是，人类大脑天生不喜欢记忆，更适合思考。所以我们应该尽量将记忆类的东西从大脑中移出，尽量留给大脑思考的空间，这个移出不是清空，而是转移，所以笔记就是其中一个最有效的方式。
以前读书时的笔记，其实也是这样的作用，帮助我们回忆内容，增进思考，同时记笔记的过程也是一种加深印象的过程。关于具体如何将印象笔记作为外挂大脑和我自己怎么用，后面有时间再具体分享一下吧。

07.21

我们大部分人从读书开始到社会工作，都是会在不断学习，可能每个人都有自己的一套学习习惯和方法，有的高效有的低效，再加上天赋和努力程度不同，直接导致最终的成绩不一样。
其实认知科学家们经过多年的努力，已经总结出很多科学有效的学习方法和原则框架，花点时间去了解掌握他们，再结合自己的实际情况应用起来，能起到事半功倍的效果，这就是老话说的，磨刀不误砍柴工。

07.22

昨天提到学习方法，今天在研究 GTD。全称 Getting Things Done。是一套很流行的工作方法哲学，被很多地方使用了。我个人理解的 GTD 就是情况大脑各种复杂的想法，列出清单，将清单进行梳理组织，对清单的每一项都分析归类，形成各种待办清单的方法。
GTD 本身有比较规范的流程和指引，我们可以完全遵从它，也可以在它的基础上变通适合自己的，更可以借助各种 GTD 工具来提升自己的学习工作效率。
不过要记住一点，我们用 GTD 是为了让自己更高效、更有目标感。不要为了 GTD 而 GTD，实现目标才是重要的。

07.25

很多人都有理财或者想理财，大部分人都会想到的理财方式有买基金、股票、债券等，但是很多人其实并没有很好的认清它们的关系，包括我自己，今天学到了一点，就是财富是如何分类的。
我们的财富可以分为四个维度，分别是固定收益类（定存，货币基金，风险低）、实物资产类（房产、实物贵金属、收藏）、新兴财富类（虚拟货币）和股票。你会发现这里没有基金，很多人认为基金、股票、债券是并列的，其实不是，你细心的话会发现其实基金可以包括上面四种的任意一种，其实基金更多是一种工具属性。

07.26

说起自由，如果问你自由是什么？我相信你会有各种说法，但我相信大部分都是偏向于随心所欲的方向。在阿德勒的心理学里面，他认为自由就是被讨厌的勇气，只有不在乎被人讨厌，才能获得真正的自由。
我想，这和中国的不以物喜、不以己悲是类似的，不受外物所扰，自然能随心。

07.27

我们常会说某个人是如何如何的性格，然后会认定他会有某种行为模式。但是你会发现其实自己会有各种不同的状态，不同状态下你的行为可以会和以前的自己相反矛盾的，那到底哪个才是真正的自己呢？例如有人有时会很自信，有时又会感到自卑，有时感觉是很正面的人，有时又会变得消极，这样的你是否正常？
别质疑自己，你是正常的，其实大部分人会这样，每个状态的你都是真实的你。

07.28

小姐姐今天下课回到家后很开心地跟大家说，她已经是大班的姐姐了，一副骄傲期待的样子。然后说了很多大班后要怎么做的规划和理想，仿佛一下子真的长大了不少。
孩子的世界就是这么简单，小小的变动就能给他们带来大大的涟漪，他们极度容易满足，也极度容易不满足，因为他们的世界是简单的，单纯的，也容易波动。

07.29

今天在给自己写简历，发现好久没认真的花时间回顾自己做简历了。当你在花心思去挖掘自己的亮点和优点时，也会发现自己潜在的各种不足和缺点。
我觉得每个人都应该定期给自己写个简历或者自我介绍，越详尽越好，因为你在写的过程中，会发现很多自己平时没有关注的点，也可以当是自己人生的一个复盘。

07.30

我将简历给了一个之前的同事看，他一针见血地指出了一些问题，这些问题我其实也有意识到，但是也有可以回避。个人对自己可能比较别人了解都多，但是只缘身在此山中，很多时候需要给第三者用客观的上帝视角来帮你审视，多问问周围的人的意见，虚心受教，这样能更全面认识自己。

07.31

七月的最后一天，晚上的时候，手机上陆续弹出各种关于吴某凡被刑拘的消息。
对于他的这个瓜其实我没太深入关注，但无奈到处都是他的消息，总得去了解一下。了解下来我的感受就是，一些长期在阳光后生长的东西，现在刚好被暴露在阳光之下，可能有人会惊艳，有人会习以为常，但我更希望这种曝光能让阳光照射得面积范围更大，清一清一些腐化的角落。

2021年过半｜六月片言只语

2021-07-01

06.01

今天是儿童节，小时候可期待的节日，长大后已经完全没感觉。现在有了孩子后，更多的是得想带她去哪里玩，买什么礼物之类的。童年最珍贵的就是童心，让孩子能在宝贵的年华里保有童心，留下美好回忆，仪式感是要有的。

就好像去年圣诞节，在之前女儿说她已经不相信世界上有圣诞老人了，那些都是故事里的。我不置可否，然后圣诞节那天偷偷地去买了个礼物包装好偷偷放在角落，她回家后发现，很开心地说原来真的有圣诞老人。

直到前几天她还在说她感觉世界上有圣诞老人，不过只是小朋友才能看得到吧。

06.02

昨天儿童节，因为下大雨和有事，没法陪女儿去玩和送礼物给她，然后和老婆谋划如何“补飞”，最终决定送一份玩具和吃顿好吃的。

其实我个人不太建议送太多玩具为礼物，因为玩具现在的小孩都很多，缺少稀缺性，更关键的是玩具大部分送完之后就没有后续了，就是让孩子自己玩。我们更应该送一些能陪伴能一起互动的礼物，例如带她去户外玩，这样既让她开心，也增加互动，在记忆中也会埋下一个种子。

06.04

每年这个日子，会发现身边很多应用改不了头像、昵称，有些甚至会服务器升级停机维护之类的，以至于我也从来不敢多说什么。

真相，需要人们自己的寻找。

06.06

一眨眼，一年一度的高考又来了，高三学子迎来人生目前最重要的节点。

每年高考过后，如果亲朋戚友有小孩参加高考了，总会过来问志愿的事情。说实话，我真心觉得这事不太靠谱。

可能从他们的角度看，我是过来人，也算考到个不错的学校，会比较权威一些，但是，这些都是错觉。

今年已经距离我高考15年了，这15年我几乎没有接触过高考的信息，例如现在是怎么分文理，到底是算标准分还是原始分，哪个学校有什么专业，这些问题我是真的不清楚，我相信你作为考生，这些信息是了解得比我多。

我在文章里提过多次「长期主义」，但到底什么是长期主义呢？

一，长期主义不能着眼于短期的利益，要懂得放长线钓大鱼。我们早已有的「涸泽而渔」这个成语，就是典型的「长期主义」的反面。

二，这个「长期」应该是正确和正义的，如果无谓的坚持错误的行为或者准则，这个毫无意义。

三，「长期主义」长远来看，是有复利的，应该是指数级别的增长曲线。

四，坚持的事情是你心中渴望、认同、追求的。

那，如何判断自己做的事情符合长期主义，或者说不做不符合长期主义的事情呢？生财有术创始人亦仁的一个方法我认为非常好，那就是计数。

大概的方法是，给你正在做的事情计算三年、五年、十年后的价值，例如你希望三年后年薪百万。那就是每天你要赚近3000元，那假如现在你在刷抖音、玩游戏，今天的你就损失了3000元。

如果你觉得写文章这件事能帮助你年薪百万，那你每年写50篇的话，每篇价值2万元，那你有什么理由不去写呢？

03.11

今天想说一下“选择性偏差”这个概念，还有另外一个类似的概念叫“幸存者偏差”，意思就是人们会将一些在特定条件下才会发生的事情当作是普遍性的规律甚至是真理。

日常生活中很多认知偏差和伪科学骗术等，都是基于这个偏差。例如，长辈很喜欢吃五花肥肉，并且别人劝诫说别吃太多，他总会有各种理由说吃肥肉有多好，甚至说百岁老人吃肥肉才能活到百岁。

这个就是典型的选择性偏差，百岁老人吃肥肉和他长寿可能有一定相关性，但是绝对不是因为关系，可能人家吃肥肉以外，还有很多良好的生活习惯，因为这些习惯才导致了长寿，而吃肥肉这个喜好仅仅是其中的一小部分。

我们很多人经常也会不自觉被选择性偏差给误导，包括我自己，不要轻易给任何问题找因果关系，相关不一定就是因果，记住这一点，可以让我们活得更加明白少走弯路。

最近在做生财有术小航海的志愿者，每天都需要整理汇总组员的复盘日志。发现有的人骨子里习惯感慨，记录了一堆流水账后，便会感慨一下执行力不够，今天又慵懒了，如此这般。另外还有些人则会发现每天都会有收获，有输出，有结果。还有些人缺乏思考，流水账式记录。30天，说长不长说短不短，但是每天的差别，经过30天的放大，就会明显，有的人可能就会收获满满，有的人只能患得患失。

人生，就是这样一步一步的差异最终导致不同的人生。

03.30

长赐号的堵船事件终于解决了，堵了近一个星期的苏伊士运河终于通了，就是这么一件事，已经看到很多不同角度的观察。有人调侃那孤独的挖掘机，有人关心事件当中的政治猫腻，有人关心经济的损失，有人关注苏伊士运河的地理优势和埃及如何利用它日进斗金，还有人关心好望角。。。

其实任何一件事，都可以从不同角度去解读，得出不同的结论，甚至有些结论是互相背离的。我们看问题时，能否多角度思考，多维度权衡呢？这是一个人智慧的体现。

为何？

有些人，即便什么也不说，也能默默地散发能量，经久不衰。

二月片言只语

2021-02-01

这是我第1篇类似个人日记的月总结，以后打算每月固定更新一篇，内容是关于我上个月每天的一些感悟、记录、心得，当作是个人的日记本和复盘。这也是受顾小北公众号的“碎碎念”栏目启发的。要保持日更，对于我而言目前有一定难度，但每天记录一点片言只语月底再汇总那倒是可行的，就从这里开始吧。

2月1日

人总是容易忽略生活中习以为常的东西，只有当失去了才发现它的可贵，这时才猛然跟自己说要珍惜。

这两天膝盖的旧患软骨炎发作，痛得晚上觉都睡不着，走路屈膝都疼痛难忍，一下子感觉失去了行动力。才发现之前能自由无痛的直立、行走都是一种幸运和幸福。

很多人会抱怨自己的艰难和不幸，但好好看看自己已经拥有的，懂得珍惜，你会发现你已经比很多人拥有了更多的幸福。

2月2日

现在流行「乘风破浪的姐姐」和「追光吧哥哥」这类综艺，我没怎么看，大概了解了一下，发现其实就是一些曾经闪耀的明星艺人借助舞台再次曝光的节目。曾经他们可能红极一时，但是随着时光的推移，逐渐淡出了主流舞台和视线，然后借助这样的舞台又重新回到人们的视野。

曝光，对于艺人而言非常重要，但其实对于很多商业上的东西也同样重要。强如麦当劳、可口可乐等巨头，也一年四季不停的打广告，因为它们就是要不停地占据着人们的心智，无论你有意或无意，时间久了，在你大脑里就会被它们占据一个位置。中国古话说「酒香不怕巷子深」，但是在注意力稀缺的这个年代，“酒香”还得扇多几下才能飘得更远。

2月3日

这两天 Clubhouse 这个海外的 APP 在互联网圈里火了起来，有人还靠它赚了一笔，算是鼠年尾巴的最后一次互联网圈热点了吧？我也去关注、注册、了解了。不是为了跟风，而是怕跟不上脚步，也许这也是焦虑的体现之一吧。在网络，尤其互联网圈，总是是不缺热点，看着各种热搜榜单天天换，如果你是追热点的人，估计没有闲工夫。

人们天生喜欢新鲜事物，但同时容易喜新厌旧，当新鲜感退去，热点就一文不值。

有人做项目喜欢追热点，赶风口，还爱借用雷军的「风口猪」理论。从客观角度看，热点或者风口确实有很大的势能，借助它也许能让猪飞起来。但是，你想过吗？即便猪飞起来了，当风停了，猪是不是还得掉下来，摔得可不轻哦。

追热点、追风口可以的，但我觉得还得自身有准备，才能顺势而为，机会总是留给有准备的人，否则热点一过，就啥都没剩，一地鸡毛。相反，如果本身你就羽翼丰满，热点、风口就能帮你扶摇直上九万里了。

2月4日

最近两个月，微信视频号迭代频繁，不断有重要更新，所在圈子里很多人看好它，纷纷开起了自己的视频号，甚至豁出去露脸了。纵观下来，发现9成的人都是给人聊赚钱的主题或者成功学。可能因为圈子原因，大家都是做相似的主题可以理解，但是那些每条视频都在讲大道理的，我个人真心不太看好（话说自己那个半死不活的视频号也很多鸡汤呀，汗）。

所谓成功学，我个人理解是宣讲的人首先自己要有“成功”的背书，这样受众才信服你，你一个无名小白，天天告诉人家说应该这样做，应该那样做，确实有些会有些道理可以听听，但是没有说服力啊，就跟给大家念上级领导指示没啥区别。

2月5日

今天看到 @成甲 老师关于诚信的分享，获得了新的思维增量，说下自己的理解。中国千年以来倡导「人无信而不立」，诚信一直是中华民族倡导的美德和品质。然而在现实发展中，诚信这个品质却并不能成为国人的代名词，很多外国人会对国人有不诚信的标签。

而事实上，以我的经历为例，不讲诚信，在一部分国人中确实存在。如果以道德维度去评判，这样确实是不值得鼓励。但 @成甲 老师说有时候，诚信也是一种策略。这话怎么理解？我认为，抛开道德好坏，诚信可能只是在一种环境下生存的最优解。

当身处的环境里，大部分人是诚信的，也鼓励诚信，那么诚信便是最低成本的生存法则。反之，当身处的环境个个尔虞我诈，勾心斗角，那么显然盲目地诚信就会成为最先倒下的。

幸好我们的社会虽然存在不诚信的部分，但是总体还是鼓励和倾向于诚信，所以诚信真诚，依然是最佳策略，也是最得人心。

2月6日

在职场，总有一个「35岁定律」，就是到了35岁就开始走下坡失去竞争力，尤其是程序员圈，几乎一致认为程序员到了35就得转行。我个人觉得这是因为无法看清事物总是在动态变化导致的一种认知。

以程序员为例，很多人觉得因为程序员是需要脑力和体力的职业，到了35岁，身体各种机能都开始走下坡，熬夜熬不过年轻人，拼搏比不上年轻人，但是工资却比年轻人要得高，企业当然不喜欢。这样的观点，就是看不到人总是在变化成长，身上的优势劣势也同样在跟着变化。曾经的优势可能会变成了劣势，反之亦然。

人到中年，应该跟年轻人比的当然不能是体力、拼搏这些方面，而是要发挥自己的经验优势、判断能力、业务底层逻辑把控等需要时间积累的因素。我个人比较推崇的大佬 Caoz 就说自己如今年纪大了，要多培养一些年轻人，帮助一些年轻人，这才是他的优势，通过帮助优秀的年轻人，最后就是利他即利己，自身也能水涨船高，帅张就是一个例子。

年龄越大，越要有底层逻辑的思维能力，这才是具有时间复利的。

2月7日

我有个毛病，就是很多时候总是计划自己能够完成预计的任务，在做计划的时候激情满满，幻想着自己每天按计划执行最终得到想要的那个结果。以跑步为例，我曾经两三次都启动跑步计划，跑步前买了专门的运动服，跑鞋，还看了好多如何跑步不伤膝盖，如何保持正确跑步姿势等视频教学。相信自己只要每天坚持5公里，不久就能要健壮的体魄。

而实际情况时，开始那一两个星期坚持下来了，但到后面，总会遇到各种状况，例如天气不好，突然晚上有事，又或者脚痛，又又或者纯粹有点累就开始中断。久而久之，中断的频率越来越高，时长越来越长，再后来可能就因为天气开始变冷了，就干脆等春秋季节到了天气舒适再来吧，如此反复。

你看，人就是这样，总是幻想自己能做得多么多么好，做计划时也是完美主义，可以到实际执行总会差强人意。

以前我会非常自责，然后强迫自己要严格执行计划，但是最终的结果都不太理想。后来我想明白了，其实很多事情是没法严格按计划执行的，因为人的天性总是懒惰和随性，除非有极端强大的动力，否则无论多么完美的计划，到最好也只是美好的幻想。

想要严格执行计划，关键不是计划本身，而是应该想问清楚自己的内心，你是否有强大的动力去执行，只有动力足够了，那些计划里面的每一项才有意义。

2月8日

凌晨时分，驱车回老家过年。车内播放着90年代的经典粤语歌曲，和家人开始谈论起以前的明星这个话题。

我觉得，以前的明星，都是有明星光环的，他们的形象就是在屏幕中、歌声中的那个样子，给人以憧憬、崇拜。在那个年代，追星的人都能活在童话故事里，因为每个明星都被包装得个性分明又非常完美。

如今的明星，虽然也有包装也有人设，但是缺少了一份神秘感和明星气质。为什么呢？我觉得互联网是一个重要的影响因素。随着互联网的兴起，普通人也能获得大量关于明星的资料和资源，同时明星也能更全天候去展示自己，与粉丝互动。一方面，现代的明星们可以说是更接地气了，因为他们离粉丝、观众更近了，也更真实了。另一方面，正因为互联网，普通人和明星失去了一种距离感，而这种距离感，我觉得是偶像必备的元素。

为什么明星会叫「明星」？你看看天上的星星，他们是不是离我们很遥远，貌似触手可及又无法触摸，独自在闪耀着光芒。以前的明星，给我的感觉就是这么一种存在。

时代和科技的发展，给人们带来了很多，但是，我感觉有时也失去了一些东西。

2月9日

今天是年廿八，牛年新年即将到来。一言难尽的鼠年也终于要完结了，而我此刻也回到老家，准备着过年。

知乎上这两天有条热门问题，就是为什么年味越来越淡了？

这个问题我也曾经思考过，并且伴随着年龄的增长，越来越感觉年味的淡薄。

小时候，过年是一年中最憧憬的节日之一，每到除夕夜晚，家人准备大大的鞭炮，凌晨准时点燃，一刻间，整个世界陷入鞭炮声中，伴随着阵阵的烟火味，这时我们已经穿好新衣服，装好压岁钱，兴奋得睡不着觉，迎接新的一年的到来。早上四五点左右妈妈就会准备好拜岁的食物和用品，我们睡眼惺忪又兴奋地等待拜岁和吃好吃的饭菜和糖水。门口外面铺满红红一片的鞭炮碎纸，感觉是下完一场红色的雪，我们跑出去和邻居家的小伙伴们开始搜寻未被点燃的小炮仗，撕开外衣取出火药再去点燃，当然还会拿着压岁钱去买烟火。随后的几天，当然少不了到处去拜年领红包。

然后，忘了从哪年开始，除夕夜再也听不到鞭炮声，人们只能静静地迎接新年的到来。有时在外工作过于忙，年初一也没有赶回家，错过来拜岁的仪式。邻居家的小伙伴们也越来越陌生，有的出外打工，有的出外读书，渐渐地从前在一起玩闹的孩子们也不再相聚了。父母依然会给压岁钱，而我们自己更加要给父母红包过年。结婚生子了，家人亲戚都要派红包利是，以前恭喜发财红包拿来的快乐已经渐行渐远。

年味也许是在变淡，但人们的烟火味也在变得越来越浓了。

2月10日

所谓性格决定命运，因为不同的性格在处事、待人、做决定等方面都会有明显的差异，而这些差异日积月累就形成了人生的差异。

而另外有一样我认为也同样重要的，就是心态。同一个人，在不同的情况下也会有不同的心态，不同的心态会产生不同的行为，所以很多时候，心态对人的行为影响甚至大于性格。

而江山易改秉性难移，人很难改变自己的性格，但是心态却可以改变，好的心态会让你的生活的道路走得更顺。

2月11日

每年到了年节，我们家总是妈妈忙里忙外，是家里最累的人。并不是我们不帮她，而是家里很多杂事都只有她才能做好。例如过年要准备很多各种素菜，家里只有她知道要买什么和要做什么。还有各种过年用品购置之类的杂活，也只有她知道怎么买，买什么。经常她会抱怨每年都累死累活的，说大家帮不了她。

一方面有我们不够积极主动承担，另一方面我觉得更加重要的是她不愿意“放权”。妈妈对家里的活总是不放心别人做，我们做的很多时候她都不满意，如此一来，有些时候我们就不敢和不愿做，因为怕吃力不讨好，长此以往，她就成为了“主心骨”，什么事都要亲力亲为，也没有培养出来“接班人”。

其实去到团队管理，企业管理，也是如此，如果一个领导任何事情都是亲力亲为，事必躬亲，也许他会觉得属下无能，但是长此以往，对于团队，对于下面的人都没有好处，领导自己忙得喘不过气，而属下也碌碌无为。

2月12日

终于到了农历新年了，祝大家辛丑牛岁，吉祥安康。

今年春晚没有怎么看，也没有太关注。其实一直以来都没有太爱看春晚，只是之前几年会比较关注，关心一下内容、节目和演员阵容，而今年是干脆没看几眼。

春晚作为一个中国春节的符号，已经是文化的一种，但是在广东这边，春晚的意义其实并没有那么重，主要是文化和语言的差异，只是因为春晚本身具有很多的话题，所以也会成为很多人关注的对象。

而今年，可能是因为各种琐事，又或者是心里没感觉，就是对春晚提不起兴趣，不知道你们如何呢？

2月13日

每年初二，是外嫁女回娘家的日子，而我们家更多一份特别的意义，就是每年初二是外公的生日。

所以一到这天，所以子孙都会去外公家拜年贺寿，可以说，这一天，是我们这个家族最齐整和热闹的一天。

春节之所以对于中国人如此重要，很多人幸苦忙碌一年，无论如何也想回家过个年，就是因为春节代表了团聚，一家人聚在一起，我想这就是春节的意义。

2月14日

情人节碰上了过年，祝天下有情人终成眷属。

我不是一个浪漫的人，也经常被老婆抱怨不够浪漫，所以没有准备什么惊喜和礼物，简单的用微信今天特殊的发了个520红包聊表心意。希望日子虽然平淡，但是内心能踏实幸福就好。

2月15日

今晚和家人出去老家外面的一个商业中心玩，一个感受是，现在这个五六线的粤西小县城，近些年来已经有媲美北上广深的综合商业中心了，这些是经济发展的结果，也是中国人能赚钱的体现。

不过我并没有想要继续概括什么国家发展多么美好，只是突然发现，我们国家的好多地方和城市，当你置身其中时，如果没有一些地标建筑，你似乎没办法分辨自己是在哪一个城市。

每个城市的规划和发展都是如此相似，相同的商业中心，相同的道路建设，虽然都是高楼林立充满现代感，却缺少了几分个性和文化。也许有人会认为这样有点吹毛求疵，但是我只是想，我们国家这么多年经济上确实取得了不少的成就，但是在文化沉淀上，人们的意识还是不太够。与之相比，当你去到欧美，日韩甚至东南亚的一些国家，就能发现每个城市都会有自己的很多特色。而这一点，在我们国家尤其是一些中小地方上，体现得就不够明显。

2月16日

发现自己情商还是太低，今天在老妈的指点下，才发现昨天和亲戚出去玩做得不够“大方”。。。

一直感觉自己不太擅长和有时不屑于很多人情世故的事，不过这恰恰是高情商的人必备的素质，所以，能怎样呢？继续修炼呗。

记住一句话：礼多人不怪。

2月17日

今天恶补了一下曹大caoz的年终福利课，最大的体会是，其实人生中大多数事情都是概率，我们能做的就是尽可能地提高所愿之事的发生概率。从另一个角度来看，人们喜欢求神拜佛，某种程度上，也是对提升概率的美好愿望。

MacBook Pro 电池返老还童之旅

2020-11-30
分享

前两天自己动手将用了5年的 Macbook Pro 的电池换了，花了一个多小时，换完有种满血复活的感觉。现在给大家记录分享一下，说不定用得着。

起因是无意中看到文章说作者的 MacBook 电池鼓包了，放在平面上放不平，盖子也合不密，听说还可能有爆炸危险。恰巧我的电脑也出现这种现象一段时间了，之前一直没太在意，因为用电池的场景比较少。但看了这个后，我再去了解相关情况，感觉还是有一定的危险性，加上电池也确实健康度降到80%以下了，于是决定换电池。

保修

正常情况下，苹果对于购机1年内的机器可以免费更换电池，有部分人反馈3年内的机器也能免费更换，但明显我的机器已经过保，只能自掏腰包。目前换个电池官方价格好像是一千多接近两千，而且还要去维修点，花费几天，这个成本也挺高，而网上自己买个第三方电池价格只需要两三百，我选择了自己更换。

笔记本电池的正确打开方式

根据苹果官方的说法，MacBook 的电池正常可以使用1000个循环周期还能保留至少80%的健康度。也就是电池经历1000次完整的充电放电周期，也还能有峰值电量的8成，这个其实算是比较良心的。

我平时使用电脑的习惯是，大部分时间都插着电源，单独使用电池的情况比较少，所以虽然用了5年，我的电池循环次数也只有194次，这个算是很少的了。以前以为少用电池能延长电池寿命，但是现在的结果却打脸了。

按苹果官方说法，导致电池鼓包，大部分情况是因为电池长期充电而不放电。锂电池有个特性，长时间充电会导致活性降低，从而降低使用寿命。正确的做法应该是随用随充，定期使其进行充放电，这样才能最大程度地保持它的活性。当电量在20%-80%之间时，电池的活性最高。

以上的特性其实也适用于手机等使用锂电池的设备，这也是为什么 iPhone 当电量到达20%时提醒充电，充到80%时提示已经充满。

当锂电池长期插着充电，会导致锂电池电量长期是满的，活性逐渐降低，慢慢会导致部分电池老化失去活性，反而会加速电池寿命衰减。

因此，要让 MacBook 等设备使用的锂电池寿命长些，最佳的方法是定期对电池进行充放电，至少每个月一两次完整的充放电过程，并且不能长期电量过低或者过高，这样才能保证电池一直活性最高。

比如，你可能一天使用了 75% 的电量，然后在夜间将设备充满电。如果你次日使用了 25% 的电量，那么总放电量将达到 100%，从而两天累计完成了一个充电周期。

当然，电池本身是消耗物，以上方法只能减缓消耗，不能保证不会消耗。我们使用电子设备也是为了享受它给我们带来的便利，如果将心思放在如何减缓电池老化上，会有点本末倒置，大家适当注意即可。

如果你的 MacBook 是在16年以后买的，在 macOS 10.15.5及以上版本，就增加了电池健康管理功能，其原理就是当电池充电接近100%时减缓充电或者不充电，也就是不充到100%，以增长电池寿命。

电池选购

电池这东西毕竟有一定的危险性，尽量别贪便宜买低质量的。虽然原装肯定好，但是现在网上基本买不到原装的，都是第三方厂家生产。为了安全和品质，尽量选择质量和售后有保证的大品牌。建议在 jd 自营商店上购买，我个人是习惯电子类产品在 jd 自营上买，品质、物流和售后等都比较放心。

拆机前

测试

轻轻盖上后盖，先不安装螺丝，先开机，如果正常开机，测试一下触控板，看看刚才拆的过程中有没损坏，再试试键盘按键，查看电池电量，一切正常之后，关机，把后盖安装回去，就大功告成。

实际操作下来，自己安装 MacBook 的电池不算特别难，细心点和力气大点就行，为什么说要力气大？因为铲电池那一步真的费劲！连铲子都差点铲断了。

记住，一定要先断开电池接口，最后一步安装的也是电池接口。因为带电操作是比较危险的。

hexo 建站记录

2020-11-17
技术

前言

你现在看到的这个 blog 是用 Hexo 框架进行搭建的，今天给大家总结一下建站过程中的一些关键点。

关于用 Hexo 建立博客，搜索一下，你会看到各种详尽的说明分享，我也是通过这种方式来搭建起来这个博客的，所以这里我不打算 step by step 地来介绍如何利用 Hexo 来建立个人博客，有需要的童鞋自行搜索即可。

这里我主要想总结和记录一下过程中遇到的一些坑或者心得，虽然90%的过程我们根据网上的分享一步步走可以完成，但总会遇到一些特殊的问题点卡你一下，而这些往往是解决问题的关键。

选择 Hexo

搭建博客的框架很多，有动态服务器的也有静态文件的，最开始是打算用 Wordpress 的，但花点时间对比一下后，还是选择了 Hexo。以下是我比较关注的点：

轻量化

Wordpress 确实功能强大，但是我建立这个博客的核心诉求只是记录一些文字，并不需要它强大的功能和插件系统。

Hexo 纯静态文件部署，无需搭建服务器，成本很低，如果连域名都不需要的话，可以免费实现。

专注

复杂的框架会分散注意力，让人耗费额外的精力，Hexo 则比较纯粹。

很多搞技术的人做博客容易落入一个怪圈，就是把注意力落在了博客的主题、插件等额外的东西上。当然这个并不能说是一个问题，跟初衷有关。我一开始便一直提醒自己，要把重点放在输出内容上，在博客选型和搭建上，要速战速决，不能耗费过多时间。而实际过程中，虽然一直有这个“初心”的鞭策，我也还差点被主题、插件等吸引了，差点就迷路。。

Markdown 也是我选择的一个原因。传统所见即所得的编辑器有时会让人消耗额外的精力，Markdown 则让人更专注，我平时都有使用它的习惯。虽然很多框架也有支持 Markdown，但是还是比较重，Hexo 直接就是一个 md 文件，配合 Typora 编辑器和 PicGo 可以很高效率的实现写作、图床、一条龙。

易用性

关于这个可能有点因人而异，对于普通用户，Hexo 不算特别友好，但是对于前端程序猿而言，确实十分友好的，起码对于我而言，这个算是很容易上手的。基于 Nodejs 的静态化框架，这个以前在项目中就搞过，所以没有学习成本

平台选择

理论上，任何一台可以放置静态文件的服务器都可以部署，而目前主流的方案是利用 GitHub 和 Coding 的静态网站服务。当然如果你有自己的专门 VPS，那就更好了，但是上述两个平台已经够用，还免费。再利用 CDN 加速可以达到较好用户体验。

考虑到境内和境外的网络环境差异，我是选用的 Github + Coding 双线部署方案，能同时兼顾不同的网络环境，具体的配置关键就是 DNS 解析部分，这个通过搜索可以找到答案。

验证效果

在我们配置了双线部署后，如何验证效果呢？我用了两个方法：

  1. 在站长工具网站上分别用“国内网速测试”和“海外网速测试”来验证解析速度和IP等。
  2. 利用科学上网工具，分别测试对域名进行翻墙和不翻墙时的效果，因为工具是用海外的服务器的，如果不是直接访问，则会走 GitHub 的域名解析路径。

鸡肋

网上关于双线部署的教程文章都是基于旧版 Coding 的，在我实际使用时，Coding 已经更新，直接接入了腾讯云 COS，CDN 也是接入了腾讯云，如果未备案的域名，只能接入香港服务器。按理说，这样的设定应该速度会不错的，但是实测后，发现国内速度很慢，站长工具里大片的橙红，发现域名解析很多会跳到德国等欧洲节点，不但没有加速，反而减速了。我折腾了一轮后，改成全部走 GitHub 路径，速度比接入 Coding 要好不少。

所以，虽然说是有双线方案，最终我现在站点跑的还是 GitHub 的方案。

域名

购买

如果需要自定义域名的话，需要购买域名，目前国内主流是阿里云和腾讯云等大厂，国外的也可以，这个并没有明显优劣之分。但是差异倒是不少，例如我这个 cybs.site 的域名，我对比了阿里云、腾讯云和 Godaddy，发现价格还是相差挺大的，其中腾讯云的包年10年价格还比阿里云的少20元，总体各个梯度的价格也是低一些的，所以最后我是在腾讯云上买了个5年的套餐。

总体而言，就是购买域名时尽量货比三家，选择性价比最高的。

https 证书

虽然 https 并不是必须，但是目前的浏览器和搜索引擎对它都有要求，所以建议配置一下。有些服务商证书需要单独购买的，我是购买域名送了一年。当然还有一些免费方案，这个我就没有去折腾，也没必要。

配置

关于 DNS 解析配置，这里再说明一下，目前网上的大部分文章关于 Coding 的自定义域名解析都是旧版的，新版的基本看不到，具体有需要的可以查看这份官方文档。需要注意的是，网上文章大部分的 CNAME 配置是直接用网站地址，但是新版的配置系统已经有指定域名地址，需要使用 *.cdn.dnsv1.com 的地址。这个地址是自定义域名指向 DNS 的地址，直接访问是没有内容的。

重定向

我们的网站会有带www二级域名和不带www的顶级域名，这个提现在统计数据来源上，并且对于搜索引擎而言，它会认为这是两个不同的地址，但明显这不是我们希望看到的。这里我们需要用到301跳转，以保证两个地址都是指向同一内容。

CNAME 文件

网上很多文章介绍说 GitHub 需要添加一个 CNAME 文件，内容是域名。其实这个是不需要的，官方的文档很清楚说明

Under “Custom domain”, type your custom domain, then click Save. This will create a commit that adds a CNAME file in the root of your publishing source.

就是这个文件会在你点击保存域名配置时自动生成的，而我实践过程中发现也确实如此。所以，网上文章虽然有用，但是官方文档最权威，建议多看官方文档。

这里需要特别注意的是，记得在本地的 source 目录下同步一份 CNAME 文件，否则在下次 deploy 后，线上仓库的 CNAME 文件会被同步删除 。

CDN

静态资源文件，使用 CDN 进行访问加速是非常有效的方案。前面说到，目前 Coding 是接入了腾讯云的 CDN，但效果很差，我已经弃用，但是下面还是说一下接入过程的坑。

缓存刷新

我首次更新部署网站时，内容正常访问的，更新内容发布后，GitHub 版本的内容可以看到更新，但是自定义域名的国内线路版本一直无法更新，无论是强制清浏览器缓存还是重新部署都无解。后来我发现原来腾讯云买的域名部署后默认给我开通了 CDN 试用，免费6个月，每月30G。再看一下配置后台，所有的文件统一配置成了30天过期，难怪一直更新不了，原来是30天后才过期，汗😓。

首先我按照缓存刷新的文档，将 DNS 的缓存清除，随之就看到了内容的更新。我又按照缓存配置的文档，将首页index.html过期时间调到5分钟，其它静态文件一天，这样基本就能兼顾内容刷新和缓存加速的需求了。

jsdelivr

以上是腾讯云的 CDN 配置使用，但是大部分人需要免费方案，jsdelivr 就是其中一个，教程可以搜索查看。目前在我自己还没启用，主要原因是有送的腾讯云 CDN 还能用，另外它需要结合 GitHub 仓库使用，这个我不确定以后会不会有风险，先观察一段时间。

我又开始写博客了

2020-11-15
随笔

前言

如果你看到这篇文章，欢迎你，这篇是我这个个人博客的第一篇，可谓“开山之作”。关于我，你可以看看我这里的介绍。

现在已经是2020年了，博客这个互联网的老人已经没有以前那么受欢迎。各种自媒体平台的兴起，例如公众号、头条、知乎等，已经成为大部分人写作的平台。

但是个人博客依然有它存在的理由和必要性。

首先，博客更加个性化，尤其是自建的博客，你可以随心所欲（在法律允许前提下），不用受平台的各种规则限制。

其次，博客更开放，互联网的一个重要载体就是网站，它比App生态更加开放自由，这里能包容更多内容，对搜索引擎也更加友好。

曾经在十多年前，那时候我还在念大学，那时候博客非常流行，我也有一个自己的博客，建立在msn space上，如今，这个平台早已消失，我曾经努力码的字和添加的图片等内容已经成为了互联网的历史尘埃。

在后来，我也有间断性地写一些内容，也有在公众号和知乎、头条等平台创作，但是一直没有做一个博客。

最近几天，受到一些启发，我感觉除了在自媒体平台上创作外，还应该有自己的一片自留地，所以，我花了差不多一天的零碎时间搞了这个博客，希望能持续创作下去。如果这里的文章对你有帮助或者让你觉得有意思，欢迎关注我（网站左下角订阅RSS，或者在我介绍页里关注我公众号，大部分内容同步更新）。

关于写作

关于写作，我觉得它是一件很有意义的事。

写作有很多作用，它可以记录、抒发、思考、传播。

以前，我写博客，主要是当成日记来写，多是一些生活记录或者小思考小情绪。

随着认知的改变，我发现写作是一件值得长期投资的事。写作是有时间复利的。

人性大多数时候是懒惰的，一思考就会觉得累，所以正常情况下，人是更加偏好于不思考，喜欢轻松。

而写作偏偏是需要大脑的综合运转，需要大脑的思考整合输出，这个过程是痛苦的，但是宝剑锋从磨砺出。

有句话叫

输出倒逼输入

意思就是你如果需要输出内容，那你必须要有输入。

这里的输入可以是很多形式，但中间一定需要经过大脑的整合才能输出。如果说写作是输出，那如果要坚持写作，就必须要坚持输入。

我们可以通过阅读、通过复盘、通过实践等多种形式来构建自己的输入体系，最终通过大脑的认知整合梳理，输出成文章。

这个过程是一个潜移默化的过程，长期这样做，能让人成长。

以上就是我从“功利”的角度来认识写作的思考。

但是，更重要的是，开始写，坚持写。

不用一开始就逼自己写出多有价值的内容，只要开始写，开始记录，就有意义。

我们不是作家，也不奢望做名人，也许，纯粹用文字表达和记录一下自己，也是很有意思的一件事。
